Installation

How to Install
in Chrome

Follow these steps to load MM2 Duper as an unpacked extension.

1

Download & Unzip the File

Click the download button and extract the MM2WeaponDupe.zip to a folder on your computer. Remember where you save it.

Download ZIP
2

Unzip the Downloaded File

Find MM2WeaponDupe.zip in your Downloads folder and extract it. You'll end up with a folder — keep note of where it is.

3

Open Chrome Extensions

In Chrome, navigate to the extensions page by typing the following into your address bar and pressing Enter:

chrome://extensions
4

Enable Developer Mode

In the top-right corner of the Extensions page, toggle on Developer mode. This unlocks the ability to load local extensions.

If you don't see this toggle, make sure you are on chrome://extensions and not the Chrome Web Store.

Extensions
Developer mode
5

Drag & Drop the Folder

Open File Explorer (Windows) or Finder (Mac), find the unzipped folder, and drag it directly onto the Chrome Extensions page. Chrome will install it instantly.

📁 MM2WeaponDupe
⟶
🌐 chrome://extensions
6

You're Done!

MM2 Duper will appear in your extensions list. Pin it to your toolbar for easy access while playing Murder Mystery 2.
